# Stm32ExtendedLib
A simple lib to encrease the Readability of Stm32 Standard Lib.Write in cpp.

# Target
项目已更新至基于2021年版标准库以及ARM CMSIS 5.8.0构建，编译请使用ARMClang v6。
理论可兼容更早的标准库以及支持cpp11的编译器。
